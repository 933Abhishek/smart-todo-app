# AI Guidance File

This file describes how AI tools were guided during the development of the Smart Todo App.

## AI Tools Used
- ChatGPT for code assistance
- AI suggestions for debugging and code structuring

## Prompting Rules
- Keep responses simple and beginner-friendly
- Prefer lightweight solutions
- Focus on Flask REST API and React integration

## Coding Standards
- Use clear and readable variable names
- Follow modular structure (backend / frontend)
- Keep API responses in JSON format
- Maintain simple REST API endpoints

## Development Constraints
- Backend must use Flask
- Frontend must use React
- Use a relational database (SQLite)

## AI Usage Scope
AI was used for:
- Generating boilerplate code
- Fixing errors during development
- Improving documentation and project structure
